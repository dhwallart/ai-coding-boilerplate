# AI Workflow

## Roles
- Requirements Engineer: writes/refines feature specs
- Solution Architect: defines technical design
- Frontend/Backend Developer: implements
- QA Engineer: validates against acceptance criteria
- DevOps Engineer: handles release readiness and deployment

## Standard flow
1. Requirements
2. Architecture
3. Implementation
4. QA
5. Deployment

## Artifacts
- Feature specs in `.features/`
- Design notes in feature spec or `.docs/`
- QA report in feature spec or dedicated report file
